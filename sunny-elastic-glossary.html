Adapter

A small, trainable module inserted into a pre-trained model to adapt it to a new task without modifying the original weights. Adapters add only a fraction of the model's parameters. Multiple adapters can be swapped in and out for different tasks while the base model stays frozen.

API (Application Programming Interface)

A standardized way for software to communicate with other software. Embedding models are typically accessed through APIs — you send text and get back vectors. The API abstracts away the complexity of running the model so you can focus on integration.

Approximate Nearest Neighbor (ANN)

A class of algorithms that find vectors close to a query vector without exhaustively comparing every one in the index. ANN trades a small amount of recall for a massive speed-up, making vector search feasible at billion-scale.

Attention Mechanism

A component in neural networks that lets the model focus on different parts of the input when producing each output token. The foundation of transformer architectures and the reason modern LLMs handle long-range dependencies so well.
